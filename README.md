# 预览增量备份助手

一个基于 Python + Tkinter 的桌面工具，用于在 **Windows** 上执行“先预览、后勾选、再增量复制”的文件备份流程。

## 功能概览

- 扫描并预览两个目录的差异（源目录 vs 目标目录）
- 左列显示“源目录到目标目录”的文件，支持勾选复制
- 右列显示“目标目录独有（源目录没有）”文件，仅预览不复制
- 支持过滤：
  - 扩展名过滤（如 `.ppt,.pptx,.pdf`）
  - 关键词过滤（路径包含）
- 支持列头排序（点击列名切换升序/降序）
- 增量复制仅从源到目标，**只增不删**
- 两种校验模式：
  - 快速模式：仅复制
  - 安全模式：复制后计算 MD5，并输出 `Verification_YYYYMMDD_HHMMSS.csv`
- 批量任务中单文件失败不中断，最后汇总成功/失败

## 目录结构（当前仓库）

- `backup_tool.py`：主程序（GUI）
- `dist/预览增量备份助手_单文件.exe`：单文件可执行程序
- `assets/backup_icon.ico`：程序图标
- `简单测试集/`：本地测试样例

## 运行方式

### 方式 1：直接运行 EXE（推荐）

双击：

- `dist/预览增量备份助手_单文件.exe`

### 方式 2：运行 Python 脚本

```bash
python backup_tool.py
```

> 说明：脚本使用标准库 + Tkinter。打包使用已安装的 `pyinstaller`（base 环境）。

## 打包命令（单文件）

```bash
pyinstaller --noconfirm --clean --windowed --onefile --name 预览增量备份助手_单文件 --icon .\assets\backup_icon.ico .\backup_tool.py
```

## 使用流程

1. 选择源目录和目标目录
2. 点击 `1. 扫描预览`
3. 查看差异列表：
   - 左列：可复制项 + 左列只读项
   - 右列：目标目录独有项（仅查看）
4. 按需输入扩展名/关键词并应用过滤
5. 调整勾选项
6. 选择校验模式：
   - 快速模式（仅复制）
   - 安全模式（复制 + MD5 对比 + CSV 日志）
7. 点击 `2. 执行增量备份`，在确认弹窗中确认执行

## 差异标签说明（界面中文）

- `源目录新增（目标缺失）`：目标没有该文件，可复制
- `源目录更新（建议复制）`：源文件较新或大小变化，可复制
- `源目标均有（无需复制）`：两端已有且未判定变化，仅展示
- `目标目录独有（源目录没有）`：仅在目标存在，仅展示

## 安全模式日志说明（CSV）

安全模式会在目标目录生成类似：

- `Verification_20260331_180000.csv`

关键字段：

- `状态`：`OK` / `FAIL`
- `相对路径`：文件相对源目录路径
- `源文件MD5(SRC_MD5)`：源文件哈希
- `目标文件MD5(DST_MD5)`：复制后目标文件哈希
- `错误信息`：失败时的错误详情

## 使用示例

### 示例 1：论文资料备份（快速模式）

- 源目录：`D:\科研资料`
- 目标目录：`E:\备份\科研资料`
- 过滤：`.pdf,.docx,.pptx`
- 模式：快速模式

效果：仅复制新增或更新的文档，目标端已删除文件不会被反向删除。

### 示例 2：项目代码备份（安全模式）

- 源目录：`D:\project`
- 目标目录：`F:\project_backup`
- 关键词过滤：`src` 或 `docs`
- 模式：安全模式

效果：复制后逐文件 MD5 比对，并输出 CSV 校验日志用于审计。

## 典型应用场景

- 科研/办公资料的日常增量备份
- 课程/实验数据分批归档
- 代码仓库外的文档与资源备份
- 移动硬盘/机械硬盘场景下的可视化备份确认

## 注意事项

- 增量判定基于文件大小和修改时间（`mtime`）
- 不会自动删除目标目录中多出来的文件
- 建议首次使用先用测试目录验证流程
- 对关键数据建议使用“安全模式”并保留日志
